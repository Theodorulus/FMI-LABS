- avem 4 cazuri:
I. a = 0 => by + c <= 0
	1. y <= -c/b
	2. y >= -c/b
II. b = 0 => ax + c <= 0
	1. x <= -c/a
	2. x >= -c/a

Din cazurile 1 pot scoate posibile limite superioare pt y, respectiv x.
Din cazurile 2 pot scoate posibile limite inferioare pt y, respectiv x.
Ptr limitele superioare, pastram minimul, iar pentru cele inferioare maximul.
Initial, cautam x_inf=-inf, x_sup=+inf, y_inf=-inf, y_sup=+inf.
Actualizam cele 4 limite pe parcurs ce primim semiplane.
=> Complexitate liniara
La sfarsit decidem in fct de cele 4 valori intersectiile:
-x_inf > x_sup (la fel ptr y) => Intersectie vida
-pt intersectia nevida:
	x_inf < x_sup && y_inf < y_sup
-pt intersectia nemarginita:
	macar una dintre cele 4 limite a ramas +-inf
-pt intersectia marginita:
	niciuna dintre cele 4 limite nu e +-inf
